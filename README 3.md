# AI Operations Studio — Client Site

Client-facing website for AI Workflow & Operations Consulting.

## Hosting on GitHub Pages

1. Create a new repository on GitHub (e.g. `your-username/ai-ops-studio`)
1. Upload both files: `index.html` and `README.md`
1. Go to **Settings → Pages**
1. Under **Source**, select `main` branch and `/ (root)` folder
1. Click **Save**

Your site will be live at:
`https://your-username.github.io/ai-ops-studio`

## To Update Content

Edit `index.html` directly — all content, colors, copy, and pricing are in one file. No build tools or installs required.
